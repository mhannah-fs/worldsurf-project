Project Name: WorldSurf

Project Description

Welcome to the WorldSurf Project page. Here you'll find links to the usability tests that we'll be performing over the next few weeks in order to optimize the user experience. These test will be performed by real world users to insure that your applications is easy to use and understand to capture as many conversions we possibly can. I'll start by creating a basic wireframe to test the flow of the app, then over the following weeks the project will progress from a low resolution greyscale interface to high resolution fully functional website with additional testing between each iteration. Please refer to the links in the table bellow to walk through one of the test or to see the results of a test.
